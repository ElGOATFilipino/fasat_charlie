# fasat_charlie

This is a macropad with the shape of the chilean satelite FASat-CHARLIE. It has 6 keys witha rotary encoder, 16 sk6812mini-e leds on the edges of its pcb and uses kmk firmware.

I made this macropad to help me open the websites I need for school instantly, record and mute! (Since I cnstantly forget the keybind for those).

## Features: 
- 16 sk6812mini-e leds, all used for as a "glow" that the macropad emanates in the dark!
- EC11 Rotary encoder /w switch for adjusting led colors and brightness
- 6 keys
- Shape of the FASat-CHARLIE satelite
